# [Spring Boot] 통합 게시판 커뮤니티
--- 

**개요**
* Spring Boot를 이용한 통합 게시판

**주요 기능**
* 회원 가입
* 자유 게시판
* 영화 검색
* 지역 검색
* 회원 정보 수정

---  
**주요 기술**
* Google OTP MFA 적용  
* 영화 검색 API 기능 사용
* 지역 검색 API 기능 사용
* 조회수, 추천수, 댓글 기능
* 세션 처리 및 권한 별 기능 구분
* 게시판 작성, 수정, 파일 업로드 기능 
* 관리자 기능 구현

-----------

**기술 스택**
* Spring Boot
* Spring JPA, Spring JDBC
* Spring Security
* thymleaf
* Mybatis
* Mysql
* AJax
* Bootstrap
* Gradle
